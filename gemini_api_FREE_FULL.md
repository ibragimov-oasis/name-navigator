---

## ⚠️ Критические правила работы с Free Tier

| Правило | Описание |
|---|---|
| **Лимиты на проект** | RPM/TPM/RPD — общий пул для ВСЕГО проекта, не на ключ. Несколько ключей в одном проекте = один пул. |
| **Три барьера одновременно** | Превышение ЛЮБОГО из трёх (RPM / TPM / RPD) = ошибка 429. Все три работают параллельно. |
| **Сброс RPD** | Каждый день в полночь по тихоокеанскому времени (UTC-8 / UTC-7 летом). |
| **Данные на Free Tier** | Google может использовать твои промпты для улучшения моделей. Для чувствительных данных — осторожно. |
| **Несколько проектов** | Хочешь больший пул — создавай отдельные Google Cloud проекты. У каждого свои лимиты. |
| **Deprecated модели** | Gemini 2 Flash и 2 Flash Lite — deprecated, скоро будут отключены. Не строй на них ничего нового. |

### 1. Gemini 2.5 Flash
`gemini-2.5-flash` | Категория: Text-out | **Стабильная**

| Лимит | Значение |
|---|---|
| **RPM** | 5 |
| **TPM** | 250 000 |
| **RPD** | 20 |
| **Контекстное окно** | 1 048 576 токенов (1M) |
| **Max output** | 65 536 токенов |
| **Ввод** | Текст, изображения, аудио, видео, PDF |
| **Вывод** | Текст |

**Возможности:**
- **Встроенный Thinking** (цепочка рассуждений) — настраивается через `thinking_config` и `max_tokens_for_reasoning`. Можно включить/выключить и задать глубину.
- **Thought Summaries** — краткое изложение хода рассуждений, полезно для отладки агентов.
- **Нативная мультимодальность** — понимает изображения, видео, аудио и PDF в одном запросе без переключения режимов.
- **Code Execution** — встроенный Python sandbox, может запускать код и возвращать результат.
- **Google Search Grounding** — поиск в интернете прямо из API (лимит 1 500 RPD через Search grounding).
- **Map Grounding** — геопространственный поиск (лимит 500 RPD).
- **URL Context** — передай ссылку, модель прочитает страницу сама.
- **Structured Output** — JSON mode, schema enforcement.
- **Function calling / Tool use** — поддержка инструментов для агентов.
- SWE-Bench Verified: ~40% (оценка на агентных задачах кодирования)

**⚠️ Важно:** RPD всего 20 — это 20 запросов в день! Используй с умом. Это лучшая стабильная модель у тебя, но лимит очень жёсткий.

**Когда использовать:** самые важные запросы дня — сложный анализ, архитектурные решения, задачи требующие глубокого reasoning.

### Fallback-цепочка для текстовых агентов

```
[Задача пришла]
    ↓
Это coding/complex reasoning?
    → ДА: gemini-3-flash-preview (RPD=20)
           RPD исчерпан? → gemini-2.5-flash (RPD=20)
           RPD исчерпан? → gemini-3.1-flash-lite-preview (RPD=500)
           RPD исчерпан? → gemma-3-27b-it (RPD=14400!)
    → НЕТ: gemini-3.1-flash-lite-preview (RPD=500)
           RPD исчерпан? → gemma-3-27b-it (RPD=14400!)
```

### Пример Python-кода — умный роутер с fallback

```python
import google.generativeai as genai
import time

genai.configure(api_key="YOUR_API_KEY")

# Ежедневные счётчики (сбрасывать в полночь PT)
daily_usage = {
    "gemini-3-flash-preview": 0,       # лимит: 20
    "gemini-2.5-flash": 0,              # лимит: 20
    "gemini-2.5-flash-lite": 0,         # лимит: 20
    "gemini-3.1-flash-lite-preview": 0, # лимит: 500
    "gemma-3-27b-it": 0,                # лимит: 14400!
}

DAILY_LIMITS = {
    "gemini-3-flash-preview": 20,
    "gemini-2.5-flash": 20,
    "gemini-2.5-flash-lite": 20,
    "gemini-3.1-flash-lite-preview": 500,
    "gemma-3-27b-it": 14400,
}

# Приоритеты для разных задач
CODING_CHAIN = [
    "gemini-3-flash-preview",         # Лучший для кода (SWE 78%)
    "gemini-2.5-flash",               # Стабильный с Thinking
    "gemini-3.1-flash-lite-preview",  # Больше RPD
    "gemma-3-27b-it",                 # Резерв (14400 RPD!)
]

GENERAL_CHAIN = [
    "gemini-3.1-flash-lite-preview",  # 500 RPD — основная
    "gemma-3-27b-it",                 # Резерв (14400 RPD!)
]

def call_with_fallback(prompt: str, task_type: str = "general",
                       use_thinking: bool = False) -> tuple[str, str]:
    """
    Возвращает (ответ, имя_модели)
    task_type: "coding" или "general"
    """
    chain = CODING_CHAIN if task_type == "coding" else GENERAL_CHAIN

    for model_name in chain:
        # Проверяем дневной лимит
        if daily_usage.get(model_name, 0) >= DAILY_LIMITS.get(model_name, 0):
            continue

        try:
            config = {"max_output_tokens": 65536}

            # Thinking доступен только в Gemini (не Gemma)
            if use_thinking and "gemma" not in model_name:
                config["thinking_config"] = {
                    "thinking_budget": 8192  # или "auto"
                }

            model = genai.GenerativeModel(model_name, generation_config=config)
            response = model.generate_content(prompt)

            daily_usage[model_name] = daily_usage.get(model_name, 0) + 1
            return response.text, model_name

        except Exception as e:
            error_str = str(e)
            if "429" in error_str or "RESOURCE_EXHAUSTED" in error_str:
                # Экспоненциальный backoff
                for wait in [1, 2, 4, 8, 16, 32, 60]:
                    time.sleep(wait)
                    try:
                        response = model.generate_content(prompt)
                        daily_usage[model_name] = daily_usage.get(model_name, 0) + 1
                        return response.text, model_name
                    except:
                        continue
                # Этот RPM исчерпан — переходим к следующей
                continue
            raise

    raise Exception("Все модели в цепочке исчерпали дневные лимиты")


# Embedding для RAG
def get_embedding(text: str) -> list[float]:
    """Использует Gemini Embedding 2 (мультимодальные)"""
    result = genai.embed_content(
        model="gemini-embedding-002",
        content=text,
        output_dimensionality=1536  # или 768 / 3072
    )
    return result["embedding"]
```

### Советы по максимизации лимитов

1. **Кешируй ответы** — одинаковые запросы не должны тратить RPD.
2. **Batch через Gemma 3** — 14 400 RPD это ~10 запросов в минуту 24 часа подряд. Используй для массовой обработки.
3. **TPM = 15K у Gemma** — короткие промпты! Gemma плохо справляется с длинными контекстами при 15K TPM.
4. **Экспоненциальный backoff** при 429 — начинай с 1s, удваивай до 60s.
5. **Search Grounding бесплатен** — используй `google_search` tool вместо self-hosted поиска где можно (1 500 RPD).
6. **Несколько Cloud проектов** — каждый проект имеет свой независимый пул лимитов.
7. **RPD сбрасывается в полночь PT** — планируй тяжёлые batch-задачи на начало дня.
8. **Gemini 3.1 Flash Lite для итераций** — 500 RPD позволяет делать 500 итераций в день. Идеально для agent loops.
9. **Gemma 3 27B = твой главный рабочий инструмент** по объёму — при 14 400 RPD можно запускать 1 запрос каждые 6 секунд круглосуточно.
10. **Live API unlimited** — голосовые агенты на `gemini-2.5-flash-native-audio-dialog` не тратят дневные лимиты других моделей.
